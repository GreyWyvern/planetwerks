  • Welcome!

    Welcome to PlanetWerks 2, a widget-sized trip through our solar system. Along the way you can learn a few surprising facts as well as seeing some amazing images of our planetary companions.

    Plenty of new features have been added, so be sure to examine the Controls and Options panels to discover all the ways with which you can interact with PlanetWerks 2.

  • Power vs. Linear Scale

    By default, PlanetWerks uses power laws to make browsing our neighbourhood a little easier. A power law scale with an exponent less than one compresses a linear scale so that far away objects appear much closer, tiny objects get larger, huge objects get smaller, and very slow objects speed up.

    You can select a linear scale for orbit speed, object size and physical distance using the Options menu. With all three options checked, you'll be able to see just how vast the solar system truly is, and how very tiny is our own little planet!
